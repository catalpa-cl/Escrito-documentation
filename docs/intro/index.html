    Getting Started

    Where should I start?#

    By looking at the examples in the examples module. The most likely use-case might be an experiment where you train a model on some training dataset and then test it on different data. You can find such an example under de.unidue.ltl.escrito.examples.basicexamples.shortanswer.WekaClassificationExample. de.unidue.ltl.escrito.examples.basicexamples.shortanswer.WekaClassificationExampleCV contains an example with cross-validation.
